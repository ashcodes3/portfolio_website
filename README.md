## ✨ Overview

A modern, premium portfolio website built with **vanilla HTML, CSS & JavaScript** — no frameworks, no dependencies. Features a sleek dark theme with indigo/violet accents, glassmorphism effects, smooth scroll-driven animations, and a fully responsive layout that looks pixel-perfect on every device.

<br/>

## 🎯 Sections

| # | Section | Highlights |
|---|---------|------------|
| 01 | **Hero** | Typewriter effect, animated stats counter, floating particles, cursor glow |
| 02 | **About** | Professional bio with a styled code-editor card |
| 03 | **Experience** | Interactive timeline — 3 roles with tags & descriptions |
| 04 | **Skills** | Category cards — PLM & Platform, Programming & AI, Leadership |
| 05 | **Education** | B.Tech details with an animated CGPA ring (9.32/10) |
| 06 | **Certifications** | 5 credential cards (McKinsey, Google, Coursera, Dassault Systèmes) |
| 07 | **Contact** | Email, phone, LinkedIn & location — all clickable |

<br/>

## 🛠️ Tech Stack

| Layer | Technology |
|-------|-----------|
| **Structure** | HTML5, Semantic elements |
| **Styling** | Vanilla CSS3 — custom properties, grid, flexbox, animations |
| **Interactivity** | Vanilla JavaScript — IntersectionObserver, requestAnimationFrame |
| **Fonts** | [Inter](https://fonts.google.com/specimen/Inter) & [JetBrains Mono](https://fonts.google.com/specimen/JetBrains+Mono) via Google Fonts |
| **Icons** | Inline SVG (Feather-style) |

**Zero dependencies. Zero build tools. Just open `index.html`.**

<br/>

## 🚀 Getting Started

```bash
# Clone the repository
git clone https://github.com/ashcodes3/portfolio_website.git

# Navigate into the project
cd portfolio_website

# Open in your browser
start index.html        # Windows
open index.html         # macOS
xdg-open index.html     # Linux
```

No installation or build step required — it's purely static HTML/CSS/JS.

## ✏️ Customization

Want to use this as a template? Here's what to change:

| What | Where |
|------|-------|
| Name, title, bio | `index.html` → Hero & About sections |
| Work experience | `index.html` → `.timeline-item` blocks |
| Skills | `index.html` → `.skill-category` cards |
| Education | `index.html` → `.education-card` |
| Certifications | `index.html` → `.cert-card` elements |
| Contact links | `index.html` → `.contact-card` elements |
| Color scheme | `styles.css` → `:root` CSS variables |
| Typewriter roles | `script.js` → `roles` array in `initTypewriter()` |

<br/>

## 🌐 Deployment

This site is designed to be hosted for **free** on GitHub Pages:

1. Go to your repo **Settings** → **Pages**
2. Set **Source** to `main` branch, root `/`
3. Your site will be live at: `https://ashcodes3.github.io/portfolio_website`

Other options: [Netlify](https://netlify.com) · [Vercel](https://vercel.com) · [Cloudflare Pages](https://pages.cloudflare.com)
